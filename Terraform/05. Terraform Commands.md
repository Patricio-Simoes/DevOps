# 05. Terraform Commands

Bellow is a list of Terraform commands.

## Initializing Terraform

```bash
terraform init
terraform init -upgrade              # Upgrade providers.
terraform init -reconfigure          # Reconfigure backend.
```

## Code Formatting

````bash
terraform fmt
terraform fmt -recursive             # Format all files recursively.
terraform fmt -check                 # Check if files are formatted.
````

## Configuration Validation

```bash
terraform validate
```

## Planing Changes

```bash
terraform plan
terraform plan -out=tfplan               # Save plan.
terraform plan -var="key=value"          # Set variable.
terraform plan -var-file="dev.tfvars"    # Set tfvars file.
terraform plan -target=resource.name     # Target specific resource.
terraform plan -destroy                  # Plan destroy.
```

## Applying Changes

```bash
terraform apply
terraform apply tfplan              # Apply saved plan.
terraform apply -auto-approve       # Skip confirmation.
terraform apply -parallelism=10     # Set parallelism.
```

## Destroying Infrastructure

```bash
terraform destroy
terraform destroy -auto-approve            # Skip confirmation.
terraform destroy -target=resource.name    # Target specific resource.
```

## Showing Resources

```bash
terraform show                       # Show current state
terraform show tfplan                # Show saved plan
```

## Outputting Values

```bash
terraform output
terraform output instance_ip         # Specific output
terraform output -json               # JSON format
```

## Refreshing State

```bash
terraform refresh
```

## Graphing Dependencies

```bash
terraform graph | dot -Tpng > graph.png
```

## Console

```bash
terraform console
```

## Force Unlocking State

```bash
terraform force-unlock LOCK_ID
```

## Tainting Resources (deprecated, use -replace)

```bash
terraform taint aws_instance.example
```

## Replacing Resources

```bash
terraform apply -replace="aws_instance.example"
```
