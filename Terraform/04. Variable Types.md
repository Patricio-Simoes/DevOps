# 04. Variable Types

Terraform supports multiple variable types, such as:

## String

A sequence of characters used for text values.

```hcl
variable "var_name" {
  type        = string
  description = "A variable of type string"
  default     = "My String"
}
```

## Number

Represents numeric values, either integers or floating-point numbers.

```hcl
variable "var_name" {
  type        = number
  description = "A variable of type number"
  default     = 0
}
```

## Boolean

Used for `true` or `false` values.

```hcl
variable "var_name" {
  type        = bool
  description = "A variable of type bool"
  default     = true
}
```

## Any

A flexible type that accepts any value (string, number, list, map, etc.).

```hcl
variable "var_name" {
  type        = any
  description = "A variable that can accept any type"
  default     = "This could be a string, number, list, or map"
}
```

## List

An **ordered collection** of elements of the **same type** (e.g., all strings or all numbers).  
Accessed using indexes (e.g., `var_name[0]`).

```hcl
variable "var_name" {
  type        = list(string)
  description = "A variable of type list(string)"
  default     = ["A String", "Another String"]
}
```

## Map

A **collection of key-value pairs**, similar to a dictionary or object in other languages.  
Keys must be unique strings.

```hcl
variable "var_name" {
  type        = map(string)
  description = "A variable of type map(string)"
  default     = {
    "key_1" = "value_1"
    "key_2" = "value_2"
  }
}
```

## Object

A structured data type that groups named attributes of specific types.

```hcl
variable "var_name" {
  type = object(
    key_1 = string
    key_2 = number
    key_3 = bool
    key_4 = string
  )
  description = "A variable of type object"
  default     = {
    key_1 = "My String"
    key_2 = 0
    key_3 = false
    key_4 = "Another String"
  }
}
```

## Tuple

Tuples are like lists but can contain different types in a fixed order.

```hcl
variable "var_name" {
  type        = tuple([string, number, bool])
  description = "A variable of type tuple"
  default     = ["My String", 42, true]
}
```
