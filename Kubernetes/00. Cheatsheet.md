## Kubectl

## General commands

```bash
#####################
# Cluster Resources #
#####################

kubectl api-resources                  # Lists all available resources in Kubernetes.
kubectl explain <resource> --recursive # Explains in detail the available resource properties for YAML definitions.

#####################
# Object Management #
#####################

kubectl get <object> -n <namespace>                    # Lists all objects of a given type, (e.g. pods), in a given namespace.
kubectl get <object> -n <namespace> -o wide            # Lists all objects of a given type, (e.g. pods), in a given namespace, with further details.
kubectl get <object> --all-namespaces                  # Lists all objects of a given type, (e.g. pods), across all namespaces.
kubectl get <object> --selector <label>=<value>        # Lists all objects of a given type, (e.g. pods), filtering the results with the specified selector.

kubectl get all -A                           # Lists all available resources across all namespaces.

kubectl apply -f <file-name>                          # Deploys an object to the k8s cluster from a YAML file manifest.
kubectl edit <object>/<object_name> -n <namespace>    # Edits the live configuration of an object, (e.g. pod).
kubectl delete <object>>/<object_name> -n <namespace> # Deletes an object, (e.g. pod), from a given namespace.

################
# Debug & Logs #
################

kubectl describe <object>/<object_name> -n namespace  # Describes the structure of an object, (e.g. pod).
kubectl logs <object_name> -n <namespace>             # Prints the logs of an object, (e.g. pod).
```

### Node commands

```bash
kubectl label nodes <node> <key>=<value>.         # Labels a node.
kubectl taint nodes <node> <key>=<value>:<effect> # Taints a node. (Pod repelling).
```

### [Pod](02.%20Pods.md#02-pods) commands

```bash
kubectl run <pod-name> --image <container-image> -n <namespace> # Deploys a pod from a container image.
```

### [ReplicaSet](03.%20ReplicaSets.md#03--replicasets) commands

```bash
kubectl scale --replicas=<desired_replicas> replicaset <rs_name> # Scales the number of pods managed by a ReplicaSet.
```

### [Deployment](04.%20Deployments.md#04--deployments) commands

```bash
kubetctl rollout status deployment/<deployment_name> -n <namespace>   # Retirms the status of a rollout.
kubectl rollout restart deployment/<deployment_name> -n <namespace>   # Restarts a deployment.
kubectl rollout undo deployment/<deployment_name> -n <namespace>      # Undo a rollout.
kubetctl rollout history deployment/<deployment_name> -n <namespace>  # Returns the list of revisions and history of rollouts.
```

### [Namespace](06.%20Namespaces.md#06--namespaces) commands

```bash
kubectl create namespace <namespace> # Creates a new namespace.
kubectl config set-context $(kubectl config current-context) --namespace=<namespace> # Changes the default namespace used in the current context.
```
