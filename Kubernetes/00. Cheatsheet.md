# 00. Cheatsheet

## Kubectl

### Pod commands

```bash
kubectl run <pod-name> <container-image> -n <namespace> # Deploys a pod from a container image.
kubectl get pods -n <namespace>                         # Lists all pods in a given namespace.
kubectl get pods -n <namespace> -o wide                 # Lists all pods in a given namespace with further details.
kubectl logs <pod_name> -n <namespace>                  # Prints the logs of a pod.
kubectl describe pod/<pod_name> -n namespace            # Describes the structure of a pod.
```

### Deployment commands

```bash
kubectl get deployments -n <namespace>                                # Lists all deployments in a given namespace.
kubectl logs <deployment_name> -n <namespace>                         # Prints the deployments of a pod.
kubectl describe deployment/<deployment_name> -n namespace            # Describes the structure of a deployment.
kubectl rollout restart deployment/<deployment_name> -n <namespace>   # Restarts a deployment.
```
