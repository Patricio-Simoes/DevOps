# 03. IAM

Iam is a **global AWS service** that manages **authentication & authorization**.

It helps control who can access AWS resources and what actions they can perform.

It is possible to create the following in IAM:

<div align="center">
  <img src="../../Assets/Images/AWS - IAM.png" alt="AWS - IAM" />
</div>

## Users

- Individual identities with credentials;
- Users are people within an organization and can be grouped.

## Groups

- Collections of users with shared permissions;
- Can only contain users, not other groups.

## Roles

- Provide temporary access for AWS services or external entities.

## Policies

- JSON documents that define permissions.
